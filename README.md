# 💸 Digital Wallet GUI App

A GUI-based Digital Wallet System built using Python, Tkinter, and SQLite.

## 🚀 Features
- Create Account
- Add Money
- Send Money
- Check Balance
- User-friendly interface (GUI)

## 🛠️ Tech Stack
- Python
- Tkinter
- SQLite

## ▶️ How to Run
1. Install Python
2. Run the file:
   python wallet_gui.py

## 📌 Project Description
This project simulates a digital wallet where users can manage their balance and perform transactions. It demonstrates GUI development and database integration.
